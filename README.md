大規模サービス技術入門にのっていた VB Code アルゴリズムの Python 版
=======

## About
整数の圧縮アルゴリズム VBcode の Python 版です！

## Usage
```bash
./ch6_encode.py in.txt > out.dat
./ch6_decode.py in.dat > out.txt
```
